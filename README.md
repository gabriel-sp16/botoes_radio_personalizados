# botoes_radio_personalizados
Uma interface de seleção de planos apresentando botões de rádio otalmente estilizados com HTML e CSS. O projeto oculta os seletores padrão do navegador e constrói uma estética moderna com tema escuro, utilizando o seletor de adjacência (+) e o pseudo-elemento ::after para renderizar um preenchimento customizado de forma dinâmica quando a opção
